# Kuky-Logistics
Courier Managament System

## Overview
Kuky Logistics is a Courier management system where users are able to track their goods in real time safely and securely. It was built to tackle the problems
of lost parcels, hold courier companies accountable and allow users peace of mind.

## Features
User Login/Sign up
Dashboard
Track parcels
Contact us
